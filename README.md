# DevBox: O Hardware da sua Carreira de Software

> *"Porque codar com café frio e setup ruim é um bug que a gente precisa corrigir."*

## Sobre o Projeto

DevBox: O hardware da sua carreira! 

Esta aplicação simula um e-commerce focado em setups, mas funciona na prática como um **Hub de Descompressão** para programadores. 
Integrei a API do **Google Gemini** para criar o exclusivo **'Oráculo do Bug'**, que gera desculpas sênior e humor em tempo real para aliviar o estresse do dia a dia. Além de contar com Design 3D Tilt (sem bibliotecas) e arquitetura de código seguro.

Role até o fim e teste o Oráculo agora!
## ⚠️ Como Invocar o Oráculo (Teste Você Mesmo!)

Por questões de segurança, a chave da IA não está no código público. Mas a aplicação está pronta para receber a sua:

1.  **Gere uma chave grátis** no [Google AI Studio].
2.  **Abra o arquivo** `js/config.js` neste projeto.
3.  **Cole sua chave** lá.
4.  **"O Oráculo está adormecido... Para despertá-lo e receber sabedoria infinita (e sarcasmo), recarregue a página e clique no botão. A mágica acontece na hora!"**
